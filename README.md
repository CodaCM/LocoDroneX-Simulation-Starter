# LocoDroneX-Simulation-Starter
This is a custom Windows batch file to start all the applications needed to use the LocoDroneX simulation.

## Applications needed
- MavProxy
- Mission Planner
- LocoPython ([Archive Here](https://archive.org/details/loco-python-3-8-0))

## Setup
The setup is quite simple. 
All you need is to edit the lines ` C:\*locopython location*\LocoPython_3-8-0\Spyder.exe ` and `C:\*locopython location*\LocoPython_3-8-0\scripts\sim_scripts\start_dronekit.bat` then replace `*Locopython location*` with the folder path for LocoPython_3-8-0
